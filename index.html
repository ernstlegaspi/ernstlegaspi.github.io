Recent Projects

Tower of Hanoi

Tower of Hanoi is a mathematical puzzle where there is three towers and n disks. In this game, the core mechanics is the same in the original puzzle game. The difference is the towers are replaced by fences and disks are replaced by wood log. Solve the puzzle and win the game!

Pig Run

Pig Run is a 3D game where you play as a pig and run from predators. It is a game where the logics of wild life is in place. You will find a food, survive for a day without getting killed by the predators. The predators are AI that chases the pig if reach a certain distance.

Day or Night

Day Or Night is a Top Down platformer. The game can switch from day or night. The player need to collect all the apples that are placed in the game. There are apples only visible to day and vice versa.

Mage

Mage is a game where you evade the fireballs that comes from the enemies. The player can die when hit by the fireball. Enemies will chase the player if met a certain distance. Fireballs are created using Niagara System

Box Raycasts

This is a game where Box AI are place in a simple plane. It moves on its own, if it detects another Box AI, it will kill itself and the other Box AI that it detected. A Niagara System FX will explode if they both died.

Psychological VR Horror Game

This is a VR horror game made with unity where the setting is in the hotel. Find clues on what happened to the hotel and report to the higher ups. Find keys and candle using your VR Controller to finish the game.

Maze Ball

This is a game where the camera itself is moving through a circular motion and the stage, which is the maze, is rotation by itself. The goal of this game is to reach the end of the maze. Physics are applied to the ball.
